ABOUT ME

Hello! I'm Illana. I'm a fullstack developer based in the San Francisco Bay Area. I'm passionate about building intuitive, mobile ready user experiences. I love creating digital tools that make complex tasks or information more accessible to the average user.

In a previous life, I was a biomedical research scientist, and later a medical writer, in frigid Boston (lobser rolls were my consolation prize). I started teaching myself how to program during grad school so that I could create new tools to help me manage the inforamtion overload of the lab. Whether it is the human body or a mobile operating system, I have always loved understanding how things work.

When I'm not building apps, I enjoy reading fantasy fiction, tech blogs, and the occassional hike along the SF cliffs. I am a strong proponent of embracing one's inner nerd!

PORTFOLIO

Goal Keeper

A task management app that allows users to set personal goals, break those goals down into actionable tasks and schedule those tasks. For each goal, users can view the percentage of completed tasks out of the total.

Created With

React · Redux · Node · Express · MongoDB · Jest · Mocha · Webpack · HTML5 · CSS3

Live Demo GitHub

My Acronym Dictionary

A utility for building a personal dictionary of acronyms/abbreviations. Supports assigning each acronym entry to a category so that a user could, for example, record abbreviations associated with web development, and simultaneously keep track of common abbreviations for gene names.

Created With

JavaScript · Node · Express · MongoDB · Mocha · HTML5 · CSS3

Live Demo GitHub

The Doc Locator

This app allows users to search a 3rd party database for doctors by location, with results displayed on Google Maps. Filters are available for physician gender and specialty.

Created With

JavaScript · jQuery · AJAX · HTML5 · CSS3
